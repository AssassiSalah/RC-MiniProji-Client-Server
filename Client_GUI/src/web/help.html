How to Log In

Fill in your username and password and then click on Login.

If you don't have an account, click on Register.

Upon verification, the main window appears.

How to Upload a File

Click on upload and import the file.

We select the file, then click open.

The upload process starts automatically.

How to Download a File

In the table on the right, you can see a list of file names. Select or click the file name.

The file name will appear in the big bar in the center. Then, press the button with the arrow up.

After pressing the button, the download progress is shown, including metrics such as total size, current size, and time estimates.
